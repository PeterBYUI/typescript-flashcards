# TypeScript Flashcards

A simple flashcard application built with React, TypeScript and Firebase.

---

## Description

TypeScript Flashcards allows users to can create create, edit, delete, and review flashcards. It uses Firebase for authentication and real-time database updates.

---

## Features

- User authentication with Firebase
- Create, edit, delete, and review flashcards
- Responsive UI (using Tailwind CSS)

---

## Tech Stack

- **Frontend:** React (with Hooks), React-Router, Tanstack Query
- **Backend:** Firebase (Auth, Firestore)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
